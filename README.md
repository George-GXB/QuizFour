# QuizFour

`input` フォルダ内の `.csv` をSQLiteへ取り込み、4択問題を出すスマートフォン対応のStreamlitクイズアプリです。

起動時にCSVをチェックし、**未取り込みのCSVファイル名だけ**をDBへ追加します。
出題時のデータ取得はCSVではなくDBから行います。

## 注意

- `input/initial.csv` の列は `DefaultTag,English,1,2,3,4,Answer,Japanese` の順番です。
- `DefaultTag` が `None` または空の場合は「タグ無し」として扱われます。

## 主な機能

### ユーザー管理
- ユーザー登録・選択（ユーザー名のみ、認証なし）
- ユーザー削除（成績も同時に削除）
- 成績シェア用メールアドレスの登録

### 出題
- **タグで絞り込み**：デフォルトタグ／ユーザータグ／システムタグ／タグ無し で出題範囲を選択
- 出題モード：
  - おすすめ（未出題を優先。不足分は正解率が低い順＋同率なら出題回数が多い順に補充）
  - シャッフル
  - 順番通り
  - 順番通り（出題回数が少ない順）
- 出題数を `10` / `20` / `30` / `50` / `全部` から選択
- 英語は常に表示、日本語はON/OFF切替

### 音声機能
- 🔊 音声モード（自動読み上げ）：問題文・選択肢をWeb Speech APIで読み上げ
- （　）部分は効果音で置換して読み上げ
- 🎤 音声で回答（いち〜よん で回答）
- 「もう一回」「問題文」「選択肢」の音声コマンド対応

### タグ管理
- **デフォルトタグ**：CSV由来（読み取り専用）
- **ユーザータグ**：自由に作成・削除・問題への付与が可能
- **システムタグ**（自動付与）：
  - 間違えた問題（1度でも間違えた問題）
  - 習得済み（5回連続正解）
  - 正解率30%未満 / 60%未満 / 90%未満
- タグ管理画面でテーブルから問題を選択してタグを編集

### 成績
- 4択回答、正誤判定、進捗表示、結果サマリー（選択肢付き）
- 成績リスト（タグ別・正解率バー付き）
- 正解率100%未満の問題をメールで送信
- 問題の報告（メール）機能：回答後・サマリー画面から問題の誤りを報告可能
- 学習成績のリセット（パスワード認証あり）

### データ管理
- ユーザーの成績・タグ情報はブラウザの localStorage に保存（端末側に保持）
- DB再構築（`initial.csv` からクリア＆再読み込み）
- 正解データのCSV反映
- `db_initializer.py` で単独でDB初期化も可能

## セットアップ

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## 起動

```bash
streamlit run app.py
```

## DB初期化（単独実行）

```bash
python db_initializer.py
```

## テスト

```bash
pytest -q
```
